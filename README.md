# Calculator-GUI-Application
# Professional GUI Calculator

## Description

A graphical calculator built with Python and Tkinter.
The application provides a clean and responsive user interface for performing basic arithmetic operations.

## Features

* Graphical user interface (GUI)
* Basic operations: addition, subtraction, multiplication, division
* Clear (reset) functionality
* Error handling for invalid input
* Dark theme interface
* Responsive button layout

## Technologies Used

* Python 3
* Tkinter

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/malekhammouda36-cloud/Calculator-GUI-Application.git
   ```

2. Navigate into the project folder:

   ```bash
   cd Calculator-GUI-Application
   ```
